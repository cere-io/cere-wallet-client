## Quick startx

1. Install dependencies:

```bash
nvm exec npm i
```

2. Copy ENV file:

```bash
cp .env.dev .env
```

3. Run the app:

```bash
nvm exec npm start
```

4. Build the project in prod mode:

```bash
nvm exec npm run build
```
